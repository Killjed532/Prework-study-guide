HTML

Basic HTML Syntax

  • Content is contained between an opening tag (<head>) and closing tag (</head>).

How to Add <> to HTML text

  • For < type &lt;
  • For > type &gt;

<head>

  • The head element contains information about the webpage.
  • In this section a developer would add the following:
    • <link> to link to an external file
      • The rel attribute specifies the relationship between the current document and the linked document.
      • The href attribute specifies the location of the external resource.
    • <meta> this would contain the author, keywords, a description, and information to assist a browser in rendering the page correctly.

<body>

  • The body element represents the visible content shown to the user.
  • The body contains three child elements. They are as follows:
    1. <header>
      • Header
    2. <main>
      • This is where the bulk oof the content on the page will go.
    3. <footer>
      • This appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation

Other Common Elements

  • <h1-6> these are used as headings the lower the number the larger the text.
  • <p> p stands for a paragraph.
  • <span> is used to create inline division.
  • <div> is used to create a division in the page.
  • <aside>
  • <br> this forces a new line in the block element

List Elements

  • <ul> this is an unordered list
  • <ol> this is an ordered list
  • <li> is a list element that is a child to <ol> <ul>

Images

  • <image> this tag is used to link to a image

    •    Be sure to fill in the attributes in the <image> of the image will not load properly.
       Some common attributes are:

    • src for the source of the image
    • alt this describes the image if the image fails to load
    • width (generally defined in pixels)
    • height (generally defined in pixels)

  • <figure> images are often nested inside a figure element
  • <figcaption> is used to put a caption on the image

Links or Anchors

  • <a> is an anchor tag. It is used to link elements on the same page and the web.
  • Common Attributes
    • href this is where the link goes. The link doesn't go between the opening and closing anchor tags.

CSS

Basic CSS

Box Styling

  • margin is the space around the border.
  • padding is the space around the element between the border and the element
  • border is the border around the element. To create a border choose a thickness, a border type, and color.
  • box-shadow is the shadow of the box.
    • /* offset-x | offset-y | blur-radius | spread-radius | color */
    • box-shadow: 2px 2px 2px 1px rgba(0, 0, 0, 0.2);

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
